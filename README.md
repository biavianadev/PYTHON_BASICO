# Introdução Python

### Importante: 
O conceito de _classe_ e _objeto_ no Python é muito importante, por isso deixarei uma explicação bem resumida sobre a definição de cada uma delas:
- Classe é um modelo que define como um objeto é construído - em outras linguagens pode ser denotada como o "tipo de dado" referente a uma variável.
- Objeto é construído a partir de uma **classe** - em outras linguagens pode ser denotada como a "variável" pela qual será atribuído algum tipo de dado.
